## About DeshiKitchen

DeshiKitchen is a modern, full-stack restaurant ordering platform built with cutting-edge technologies. It combines a stunning visual experience with robust functionality for both customers and restaurant operators.

### Customer Features

- **Menu Browsing** - Browse dishes by category with search, filters, and spice level indicators
- **Shopping Cart** - Full cart management with sound feedback and quantity controls
- **Secure Checkout** - Stripe integration with Apple Pay, Google Pay, and card support
- **Order Tracking** - Real-time status updates from kitchen to delivery
- **Coupon Codes** - Apply discount codes at checkout

### Admin Dashboard

The comprehensive admin panel includes:
- Real-time order management with status updates
- Menu CRUD with search, categories, and availability toggles
- Coupon management with percentage/flat discounts
- Customer list with order history and spending stats
- Revenue analytics and popular dish reports
- Restaurant settings (delivery fee, tax, hours)

### Technical Excellence

Built with a modern stack including Next.js 16, TypeScript, Firebase Auth + Firestore, Stripe Checkout with webhooks, Zustand state management, Framer Motion animations, and React Three Fiber for the immersive 3D experience.
